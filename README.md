# Major Projects

## Soft Robotics Shape Sensing Research

This research explores how flexible robotic structures can detect their own shape using embedded sensing systems.

Traditional rigid robots rely on joint encoders to determine configuration, but soft or continuum robots deform continuously. This makes it difficult to determine their shape and position without embedded sensing.

Our work investigates methods for reconstructing the shape of flexible robotic spines using strain-based sensing techniques.

Current research includes:

- Studying fiber-optic sensing approaches such as **Fiber Bragg Gratings (FBG)**  
- Exploring flexible sensing methods using **conductive elastomers**  
- Designing sensing architectures for **continuum robotic structures**  
- Evaluating manufacturability and cost tradeoffs of different sensing methods  

These sensing techniques have applications in:

- surgical robotics  
- minimally invasive medical devices  
- continuum manipulators  
- soft robotic locomotion

## UAV Simulation and Autonomy Research

A project focused on understanding the dynamics and safety behavior of autonomous drones.

This work involves building simulation environments that model UAV motion and allow testing of failure scenarios.

The simulation environment includes:

- **6-DoF drone dynamics modeling**  
- control system testing  
- Monte Carlo simulation of system disturbances  
- analysis of failure conditions  

Example failure scenarios studied include:

- motor failure  
- payload variations  
- wind disturbances  
- sensor drift  

The goal is to evaluate how UAV systems behave under adverse conditions and develop logic for **safe recovery and emergency landing**.

## Blindspot

Blindspot is a wearable assistive vision system designed to help visually impaired users better understand their surroundings.

The system combines an embedded camera with a lightweight computer vision model to detect objects in the user’s environment and communicate relevant information through an audio interface.

The goal of the project was to create a low-cost system capable of performing real-time scene understanding using lightweight hardware.

Key components of the system include:

- A **camera module mounted on a headset** to capture the user's field of view  
- An **embedded computer vision model** used to detect objects in the scene  
- A communication pipeline that sends captured images to a processing server  
- Audio feedback that informs the user about detected objects  

The system was designed to prioritize **fast inference and low latency** so that users receive information quickly enough to be useful in real-world environments.

Technologies used:

- embedded camera module  
- computer vision object detection  
- React Native interface  
- server-based inference pipeline
